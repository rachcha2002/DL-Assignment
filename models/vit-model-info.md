# Vision Transformer (ViT) Model

## Model Specifications
- **Architecture:** Vision Transformer Model
- **Pretrained on:** ImageNet
- **Total Parameters:** ~85.8M
- **Dataset Split:** 80% Training / 20% Testing

## Performance
- **Accuracy:** 97% (EuroSAT dataset)
- **Training Status:** Completed

## Usage
This model is fine-tuned for satellite image classification on the EuroSAT dataset and can be applied to land use and land cover classification tasks such as identifying urban areas, forests, crops, and water bodies.
